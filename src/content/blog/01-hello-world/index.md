Hey there! I'm Ivan — a Python backend engineer who enjoys building things that work, scale, and don't wake anyone up at 3am.

## What I do

I spend most of my time designing and building backend systems: REST APIs, async services, data pipelines, and whatever else needs to run reliably in the background while the frontend gets all the glory.

My go-to stack is Python with FastAPI, PostgreSQL, and Docker — but I'm always curious about what else is out there.

## Why this blog

I tend to learn best by writing things down. This blog is where I'll share:

- Things I figured out the hard way
- Tools and patterns I find useful
- Side projects and what I learned building them

No fluff, no filler — just things I'd want to read myself.
